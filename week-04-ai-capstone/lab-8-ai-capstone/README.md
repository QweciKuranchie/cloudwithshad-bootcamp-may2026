## 🎯 What you'll build

**This is the portfolio piece.** Build a complete production AI app: a customer feedback form (S3) → API Gateway → Lambda → Comprehend → DynamoDB → admin dashboard (S3). Six AWS services chained together. You'll show this on your LinkedIn, your CV, your GitHub.

## 📋 Prerequisites

- [ ] All previous labs complete
- [ ] Comfortable with Lambda, API Gateway, DynamoDB
- [ ] Time blocked: this needs proper focus, not 10-minute chunks

## 🛠️ AWS Services used

- **S3** (× 2) — frontend form + dashboard
- **API Gateway** — HTTP endpoint for form submissions
- **Lambda** (× 2) — process feedback + return dashboard stats
- **Comprehend** — sentiment + entities
- **DynamoDB** — feedback storage
- **IAM** — role-based permissions
- **CloudFront** (optional bonus) — CDN + HTTPS
