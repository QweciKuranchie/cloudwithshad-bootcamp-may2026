## 🧠 Concepts we'll cover

- **AWS AI service overview**: Textract, Comprehend, Rekognition, Polly, Translate, Transcribe
- **Textract** — extracting text + tables + forms from any document
- **Comprehend** — sentiment, entities, key phrases, language detection
- **Rekognition** — object detection, face analysis, content moderation
- **AI service chaining** — combining multiple services to solve real problems
- **API Gateway** — HTTP endpoints for Lambda functions
- **DynamoDB** — serverless NoSQL for flexible JSON data
