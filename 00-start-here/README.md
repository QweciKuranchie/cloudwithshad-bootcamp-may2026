## 🚦 60-second smoke test

Run this in your terminal to confirm everything is installed and working. If all four commands return a version number, you're good for Week 1.

```bash
python --version       # should show Python 3.10 or newer
git --version          # should show git version 2.x or newer
aws --version          # should show aws-cli/2.x
code --version         # should show VS Code version
```

If any of them fails with "command not found", go back to the relevant setup doc.

## 🆘 Stuck on setup?

**Read this first**: most setup issues fall into 3 categories:
1. **AWS account stuck "verifying"** — see [aws-account-setup.md](./aws-account-setup.md) for the Telecel/MTN troubleshooting steps
2. **Python or pip command not found** — usually a PATH issue. See [python-vscode-setup.md](./python-vscode-setup.md)
3. **Git SSH key not working** — see [github-setup.md](./github-setup.md), the SSH section

If those don't help: post a screenshot in the WhatsApp group. **Use the group, not DM** — likely 5 other people have the same problem and your question helps everyone.
