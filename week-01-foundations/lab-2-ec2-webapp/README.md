# 🛠️ Lab 2 — EC2 Web App

> **Type**: Build-on-your-own
> **Time**: ~75 minutes
> **Difficulty**: Beginner / Intermediate
> **Cost**: Free (within free tier)

---

## 🎯 What you'll build

Launch an EC2 instance, SSH into it, install Nginx, and serve a custom webpage. This is your first taste of "real" cloud — managing a Linux server in the cloud the same way pros do.

## 📋 Prerequisites

- [ ] Lab 1 complete (you should know your way around the AWS Console now)
- [ ] You can SSH from your laptop (Terminal on Mac/Linux, Git Bash on Windows)
- [ ] You have a fresh EC2 key pair ready (we cover this in the walkthrough)

## 🛠️ AWS Services used

- **EC2** — the virtual machine
- **VPC default** — networking (no custom VPC needed yet)
- **Security Groups** — your firewall
