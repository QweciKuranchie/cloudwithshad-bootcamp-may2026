## 🎯 What you'll build

Deploy a personal portfolio website to AWS — accessible from anywhere in the world. We'll use S3 for hosting and CloudFront as a CDN to make it fast across West Africa and globally.

## 📋 Prerequisites

- [ ] AWS account created and signed in
- [ ] You know which AWS region you picked (e.g., `eu-west-1`)
- [ ] You have an `index.html` you want to host (or use the starter we provide)

## 🛠️ AWS Services used

- **S3** (Simple Storage Service) — hosts the static files
- **CloudFront** — a CDN that serves your site globally + adds HTTPS
- **IAM** — for bucket permissions
