## 🗺️ What you'll build

By the end of 4 weeks, you'll have shipped:

- 🏠 **A live portfolio website** on S3 + CloudFront
- 🖥️ **A web server** running on EC2 with Nginx
- 🔐 **A two-tier app** with VPC, public/private subnets, EC2, and RDS
- ⚡ **A serverless image resizer** with S3 + Lambda
- 📈 **An auto-scaling app** with ALB and Auto Scaling Groups
- 🔄 **A CI/CD pipeline** that auto-deploys GitHub pushes to AWS
- 🧠 **A Smart Document Analyzer** with Textract + Comprehend
- 🏆 **A capstone AI Customer Feedback Hub** — your portfolio piece
- 🎤 **A group project** demoed live to the cohort (5 mini-awards up for grabs)

All of it lives on your GitHub. All of it is shareable on your LinkedIn. All of it makes your CV stronger.
