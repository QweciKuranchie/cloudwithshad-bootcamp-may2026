## 🎓 Cloud Practitioner Exam Prep

The bootcamp gets you ready for the exam. To actually sit and pass it:

- 📖 [Official exam guide (CLF-C02)](https://aws.amazon.com/certification/certified-cloud-practitioner/)
- 🎯 cloudwithshad practice quizzes (link coming soon — Shadrack&apos;s own question bank)
- 📺 freeCodeCamp's 14-hour AWS course (free on YouTube)
- 💵 Exam cost: **$100** (taken online or at a Pearson VUE center)
- 🏆 Pass mark: **700 / 1000** (~70%)

### Bootcamp coverage vs the exam

| Exam domain | Bootcamp coverage |
|-------------|-------------------|
| Cloud concepts (24%) | ✅ Weeks 1-2 |
| Security & compliance (30%) | ✅ Week 2 (IAM, SGs, NACLs) + cherry-picked across |
| Technology & services (34%) | ✅ All 4 weeks |
| Billing & support (12%) | ✅ Week 1 + scattered tips |

The bootcamp gets you to ~85% exam readiness. The last 15% comes from practice tests.

## 🗺️ Glossary

| Term | Means |
|------|-------|
| **AMI** | Amazon Machine Image — the template an EC2 boots from |
| **ASG** | Auto Scaling Group |
| **AZ** | Availability Zone — a data center within a region |
| **CIDR** | Classless Inter-Domain Routing — IP notation like `10.0.0.0/16` |
| **IAM** | Identity and Access Management |
| **NACL** | Network Access Control List — subnet-level firewall |
| **SG** | Security Group — instance-level firewall |
| **VPC** | Virtual Private Cloud — your isolated network in AWS |
